# Wheel of Life / Audit de claritate

Proiect React + Vite, pregătit pentru Netlify.

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Setări Netlify

- Runtime: Node
- Base directory: gol
- Package directory: gol
- Build command: `npm run build`
- Publish directory: `dist`
